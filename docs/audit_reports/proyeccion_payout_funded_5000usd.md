# 💰 Fase de Cosecha: Proyección Cuenta Fondeada ($5,000)
**Plan de Rentabilidad Mensual Sostenible (Post-Challenge)**

## 1. El Cambio de Mentalidad (Preservación)
Una vez fondeado, el objetivo ya no es "pasar rápido", sino cobrar cada mes ("Payout"). Bajaremos el riesgo un **33%** respecto a la fase de challenge para garantizar que nunca toquemos el Daily Drawdown.

### Configuración de Cosecha:
- **Lotaje Único**: **0.10 Lotes** (por cada 0.01 de la cuenta pequeña).
- **Target de Basket**: **$50.00 USD**.
- **Piso de Seguridad (Trailing)**: Bloqueo de ganancias en +$15.00 tras llegar a +$30.00.

---

## 2. Proyección de Resultados (Simulación Operativa)

| Periodo | Baskets Estimados | Ganancia Bruta | Profit Split (80%) |
| :--- | :--- | :--- | :--- |
| **Diario** | 5 | $250.00 | **$200.00** |
| **Semanal** | 25 | $1,250.00 | **$1,000.00** |
| **Mensual** | 100 | $5,000.00 | **$4,000.00** |

---

## 3. Seguridad y Retiros
- **Drawdown Máximo Estimado**: -$150.00 (Solo el 3% de la cuenta). Muy lejos del fallo de 5%.
- **Recuperación**: Con el sistema de 3 pares, cualquier pérdida se recupera en **2-3 baskets**.

### 💡 Veredicto del Científico de Datos
El bot está diseñado para la **Consistencia**. En una cuenta de $5,000 fondeada, un ingreso neto para ti de **$4,000 USD mensuales** es el escenario base si el bot mantiene el rendimiento de hoy. 

**Estrategia Recomendada**: Retirar ganancias quincenalmente para asegurar el capital antes de decidir subir el lotaje a 0.20.
