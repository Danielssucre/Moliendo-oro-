# SKILL: Sistema L-H-N — Resolución Universal + Auditoría

## ¿Qué hace esta Skill?

Cuando el problema es complejo y requiere explorar múltiples hipótesis antes de llegar a una solución, este protocolo te guía para:

1. **Generar hipótesis** (HIVE de especialistas)
2. **Destruir las débiles** (Crisol de Némesis)
3. **Sintetizar las que sobreviven** en un output final

## Protocolo L-H-N: Instrucciones

### PASO 1 — MEMORIA OPERATIVA
Antes de responder, abre un "Diario de Intentos" que registra **cada rama explorada** para evitar bucles.

```
## 📓 DIARIO DE INTENTOS
[RAMA 1]: <Hipótesis o acción>
[PROCESO]: <Qué se investigó, qué herramientas se usaron>
[ESTADO]: ✅ SOBREVIVE / ❌ DESCARTADO
[VERIFICACIÓN NÉMESIS]: <Fortaleza o razón del descarte>
[CONTROL DE OLVIDO]: Registrado. Disponible para consulta futura.
```

### PASO 2 — CONVOCATORIA HIVE
Convocar especialistas virtuales según el dominio del problema:
- **Dominio Quant/Trading:** Experto en Market Microstructure, Quantitative Finance
- **Dominio ML/RL:** Deepmind-level RL researcher
- **Dominio Sistémico:** Arquitecto de sistemas complejos
- **Dominio Critico:** El Némesis (destructor de ideas débiles)

### PASO 3 — CRISOL DE NÉMESIS
El Némesis interroga cada idea con:
- ¿Esta solución falla bajo estrés real?
- ¿Existe una alternativa más simple que logre lo mismo?
- ¿Hay un escenario donde esto cause más daño que bien?

Si no puede responder las 3 → ❌ DESCARTADO
Si sobrevive las 3 → ✅ SOBREVIVE

### PASO 4 — SÍNTESIS FINAL
Construir la solución únicamente desde las ramas marcadas ✅ SOBREVIVE.
Documentar la síntesis como output final reconstruido.

## Cuándo usar esta Skill

- Preguntas de diseño arquitectónico complejo
- Análisis de riesgo con múltiples caminos posibles
- Cualquier pregunta donde la primera respuesta puede ser incompleta o sesgada
- Decisiones de alto impacto sobre el bot de trading

## Nota de Seguridad

Si el proceso se vuelve complejo, consultar el Diario de Intentos anterior antes de generar nuevas ramas. Nunca explorar una rama ya marcada como ❌ DESCARTADO.
