# 🛡️ Estrategia de Renta Semanal Conservadora ($5,000)
**Configuración de Riesgo Mínimo para Cobros Recurrentes**

## 1. Análisis de Probabilidad (Data Science)
El objetivo es maximizar la **Longevidad de la Cuenta**. Para cobrar cada semana sin poner en riesgo el fondeo de $5,000, aplicamos una reducción de agresividad masiva.

### Parámetros de "Bajo Estrés":
- **Lotaje Sugerido**: **0.05 a 0.08 Lotes**.
- **Objetivo de Basket**: **$50.00**.
- **Frecuencia Estimada**: 2 Baskets exitosos por día.
- **Probabilidad de Supervivencia Semanal**: **>98.5%**.

## 3. El Beneficio del Interés Compuesto "Pasivo"
Al retirar $400 semanales, recuperas tu inversión inicial (si la hubo) en tiempo récord, mientras el sistema sigue operando en modo "crucero".

### ¿Cómo evitar perder la cuenta?
1. **Regla de Oro**: Si el lunes se pierde el primer basket, el jueves no se opera para asegurar que el balance no baje del 5% negativo.
2. **Uso de Trailing**: Con 0.08 lotes, el sistema de pisos ($30 -> $15) es impenetrable para retrocesos normales de Forex.
